# Brand Seed

## Editorial name

`Agents Company by Escalona Labs`

## Technical short name

`agents-company`

## Approved visual direction

The current approved seed uses:

- a hexagonal symbol
- a connected node graph inside the symbol
- a dark blue primary tone
- a bright sky-blue accent
- a clean light background
- a wordmark with strong weight for `Agents Company`
- a lighter supporting line for `by Escalona Labs`

## Initial palette

These values are the initial working palette for product surfaces until the brand guide is refined:

- `brand.navy = #17355f`
- `brand.sky = #71c9ff`
- `brand.blue = #356fa8`
- `brand.ice = #eef6fb`

## Usage rules for v0

- Prefer the hexagon symbol as the core product mark
- Keep the symbol and wordmark paired in formal product contexts
- Use the navy tone for primary text and product chrome
- Use the sky accent for active states, outlines, or network motifs
- Avoid introducing additional brand colors before `AC-006`
