# Security Policy

## Scope

This repository contains the governance, backlog, and future implementation surface for Agents Company by Escalona Labs.

## Reporting

If you discover a security issue, do not open a public issue with exploit details.

Instead, report it privately to:

- `escalona@escalonalabs.com`

## Expectations

- Include reproduction details, affected paths, and impact
- Avoid sharing secrets or live credentials
- Prefer the smallest proof needed to demonstrate the issue

## Handling

- Security-impacting work should be tracked in GitHub with restricted detail when appropriate
- High-risk findings should produce a follow-up remediation issue with validation criteria
